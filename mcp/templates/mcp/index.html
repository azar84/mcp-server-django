Available Tools

general_get_server_status

Check if the MCP server is working and get basic server information. Use when user asks "is the server working", "test connection", or "server status".

Required Scopes:

none

Request token with: [] (no scopes required)

bookings_get_staff_availability

Get staff availability from Microsoft Bookings for a 7-day window

Required Scopes:

booking ms_bookings

Request token with: ["booking", "ms_bookings"]

general_current_time

Get current server time in ISO, timestamp, or human-readable format

Required Scopes:

basic

Request token with: ["basic"]

general_calculator

Perform basic mathematical calculations safely

Required Scopes:

basic

Request token with: ["basic"]

general_get_timezone_by_location

Get both Windows and IANA timezones for any city or location worldwide

Required Scopes:

basic

Request token with: ["basic"]

Special: Saskatchewan cities correctly return Mountain Standard Time

Scope-based Access Control

Tools are filtered based on scopes assigned to your token. Request specific scopes based on the tools you need:

  • Basic tools: ["basic"]
  • MS Bookings: ["booking", "ms_bookings"]
  • File operations: ["files"]
  • Web requests: ["web"]
  • System access: ["admin"]

API Testing

List Available Tools

curl -X POST {{ server_url }} \ -H "Authorization: Bearer your-token" \ -H "Content-Type: application/json" \ -d '{"jsonrpc":"2.0","id":"1","method":"tools/list"}'

Test Server Status (Connection Test)

curl -X POST {{ server_url }} \ -H "Authorization: Bearer your-token" \ -H "Content-Type: application/json" \ -d '{ "jsonrpc": "2.0", "id": "1", "method": "tools/call", "params": { "name": "general_get_server_status", "arguments": {} } }'

Call MS Bookings Tool

curl -X POST {{ server_url }} \ -H "Authorization: Bearer your-token" \ -H "Content-Type: application/json" \ -d '{ "jsonrpc": "2.0", "id": "1", "method": "tools/call", "params": { "name": "bookings_get_staff_availability", "arguments": { "startLocal": "2025-09-20T14:00:00", "timeZone": "Eastern Standard Time" } } }'

Scope Reference

Domain Scopes

booking Booking & scheduling functionality

basic Basic utility operations (time, calculator, timezone)

Provider Scopes

ms_bookings Microsoft Bookings API

calendly Calendly integration

google_calendar Google Calendar API

stripe Stripe payment processing

Action Scopes

write Create and modify operations

Required for booking appointments, creating records, and data modifications.
